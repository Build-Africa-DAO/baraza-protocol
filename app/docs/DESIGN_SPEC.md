# Baraza Design Spec

This public design-spec placeholder intentionally omits BRZA pricing, sale planning, allocation tables, vesting schedules, and liquidity-launch material pending counsel review.

Design work in this repository should focus on community onboarding, dues collection, treasury visibility, governance flows, and member participation. If token-distribution screens or sale-oriented copy need to return later, they should be reintroduced only after legal review.
